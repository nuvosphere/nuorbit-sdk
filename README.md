# NuOrbit SDK

Standalone version of the NuOrbit frontend toolkit. It contains:

- `NuorbitSdk`: headless multi-framework orchestrator for NuOrbit checkout flows.
- `launchNuorbitCheckout`: optional popup UI launcher.
- Config helpers (chains, direct receivers, provider call typing) you can override per app.

## Development

```
npm install
tsc --noEmit
```

Publish this repo to npm or consume it directly in other projects.
